## วิธีใช้ Ground Truth

1. **Copy ข้อความ** ระหว่าง ``` ``` ของแต่ละแบบฟอร์ม
2. **Paste** ลงในช่อง Ground Truth ใน Dashboard
3. **อัปโหลดเอกสาร** ที่ตรงกับแบบฟอร์ม
4. กด **Run All OCR Engines**

**หมายเหตุ:** 
- Ground Truth นี้เป็น template ทั่วไป
- ถ้าต้องการความแม่นยำสูงสุด ควรพิมพ์ข้อความจริงจากเอกสารที่ scan
